# Doraemon

Doraemon's pocket, there is many kinds of basic tools~ powered by SpringBoot.

- based on SpringBoot 2.x
- common utils
- REST API
- Swagger-UI
- mybatis-plus
- OKhttp 3.x
- Redisson
- JMockit
- Netty

### Environment

- JDK 8
- MySQL 5.7
- Redis 4/5
- RocketMQ 4.2
- Dubbo 2.6.4
- Zookeeper 3.5.5

### Build

```java
$ mvn -DskipTests=true clean package
```

### Run

```java
java -jar doreamon.jar
```
